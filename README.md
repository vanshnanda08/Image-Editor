# 🖼️ Image Editor

A modern, responsive image editor built using **HTML**, **CSS**, and **Vanilla JavaScript**. The application allows users to upload an image, apply real-time filters using the HTML5 Canvas API, use preset effects, reset edits, and download the edited image.

## ✨ Features

- 📤 Upload images from your device
- 🎚️ Real-time image adjustments
  - Brightness
  - Contrast
  - Saturation
  - Hue Rotation
  - Blur
  - Grayscale
  - Sepia
  - Invert
  - Opacity
- 🎨 One-click preset filters
  - Vintage
  - Black & White
  - Vivid
  - Cool
- 🔄 Reset all filters
- 💾 Download the edited image
- 📱 Responsive and modern UI
- ⚡ Live preview using the HTML5 Canvas API

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- HTML5 Canvas API

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/image-editor.git
```

2. Navigate to the project folder

```bash
cd image-editor
```

3. Open `index.html` in your browser.

No additional setup or dependencies are required.

## 📸 How to Use

1. Click **Choose Image**.
2. Select an image from your computer.
3. Adjust the sliders to apply filters.
4. Apply preset effects with a single click.
5. Click **Reset** to restore default values.
6. Click **Download** to save the edited image.
